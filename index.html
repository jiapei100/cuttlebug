Welcome to Cuttlebug

Cuttlebug is an IDE for embedded systems written entirely in python. It aims to provide a compelling enhancement to GDB, to provide simple, modern IDE features, and maintain a set of specialized tools targeted specifically at work on embedded systems.

If you're on windows, a handy installer is provided. If not, Cuttlebug runs great on linux, but from source only at the moment. If you want to run from source, the instructions and dependencies are still hosted at cuttlebug.org, but will be moving to github soon.

Support or Contact

If you're having trouble with cuttlebug, or have any questions, you can contact me directly, or check out the source code. Documentation and code cleanup is pending, now that the project has been moved to github. Expect more changes and excitement in the coming weeks!
